# Shopping list Electron Application

Cross platform shoppinglist desktop application built on electron

Tutorial: https://github.com/bradtraversy/electronshoppinglist/

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To run electron

```sh
$ npm start
```

### To Package & Build

For Windows

```sh
$ npm run package-win
```

For Mac

```sh
$ npm run package-mac
```

For Linux

```sh
$ npm run package-linux
```
